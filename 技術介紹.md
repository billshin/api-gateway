# API Gateway 技術介紹

以下介紹每個節點使用計術簡介

### Traefik

1. 反向代理 
    - 文件自動化配置
    - docker自動掃描    
    - 設定後無須指令重新載入

2. Middleware 介入管控
    - 流量限制
    - 認證機制(Auth API)

3. Load Balancer
    - 對於反向代理目標,提供負載均衡
    - 健康檢查 , 輪詢API並標示健康狀態


### Prometheus

1. 自動收集 **聚合型** Metrics 資料(定時pull)
    - scrape_interval 可設定多久要監控(預設15s,也是常見的值)
2. 使用 **PromQL** 查詢資料
